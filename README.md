## About

My portfolio site, built using [Tailwind](https://tailwindcss.com) and [Alpine.js](https://alpinejs.dev).

## Building

### Prerequisites

To build this project you'll need Node v18.6.0 and PNPM installed.

### Commands

Install dependencies

```
pnpm i
```

Run dev server

```
pnpm dev
```

Build the assets

```
pnpm build
```
