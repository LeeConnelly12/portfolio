Projects

These are some of the projects I have worked on.

A to Z

A to Z is a web app where users can login to post a jobs for interpreters to complete. interpreters can then login to accept or reject jobs they have been matched with.

Stadium

Stadium is a Wordpress website that was part of a client's rebranding process. The new website also required an integration with talentvine job's API to retrieve available vacancies.

Gas Data

Gas Data is a Wordpress website. The client wanted a more modern and responsive website along with some animations.

Club Global

Club Global is an app where users can login to book tables or buy tickets for events at clubs. Users can also check the availablity of a date by selecting from a calendar.

Plenish

Plenish is a company that sells healthy plant-based drinks. The website was built using Wordpress and Woocommerce.

Fitzgerald

Fitzgerald is a civil engineering contractor based in Birmingham, the client wanted a website in order to increase their online presence.
